Core Mechanical Design Concepts

The industrial design process in reputed industries and MNCs involves the following steps:
# Scope/Customer Inputs
# Concept Generation
# Generation of 3D Assemblies and Parts
# Simulation, Testing and modifications based on results
# Generation of Technical Drawings & Bill Of Materials

Scope/Customer Inputs

This is the first step in industrial design of any machine, mechanism or structure. The company gets Inputs from the client about what features and specificatins are required in the product to be designed and manufactured.

Concept Generation

After receiving the client inputs, the initial concept of the project is generated involving the below steps:
#Selection of required machine components and checking their suitability in machine i.e. Bearings, Ball Screws, Rails etc.
# Selection of raw materials and type of manufacturing components such as machine base, column, brackets, slides ets.
# Creation of Machine Layout

Applications of CAD Tools

After Concept generation 3D CAD Modeling Tools are used to create assemblies and parts

Simulation and Testing

After the features and specificatins of the product are decided it is tested using simulations softwares to ensure that it should work properly after manufacturing.
This process involves
# Simulation Concepts
# Factor of Safety analysis using simulation tool.

Technical Drafting

After Successful simulation and testing the frication drawings are created for manufacturing of the various parts of the product.
This process involves
# GD&T
# Limits, fits and tolerances
# Rules for dimensioning
# Features and their applications
# Roughness grades
